# Python Foundations — Phase 1

## Week 2 Project: ATM Simulator

A command line ATM simulator built with Python.

### Features
- Check balance
- Deposit money
- Withdraw money
- Input validation (negative amounts, overdraft)
- Starting balance: ₹10,000

### How to run
python3 atm_simulator.py

### Concepts used
- while loops
- if/elif/else
- break
- f-strings
- Input validation
